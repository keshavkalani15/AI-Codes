# 1. Student Management

## Objects

### Student

* Student Name → Text
* Email → Email
* Phone → Phone
* Address → Text Area
* Course → Lookup(Course)

### Course

* Course Name → Text
* Fees → Currency
* Duration → Number

### Marks

* Student → Lookup(Student)
* Subject → Text
* Marks → Number
* Result → Picklist

#### Result Values

* Pass
* Fail

---

# Relationships

* Student → Course
* Marks → Student
