# Employee Management Application using Apex in Salesforce

## 🟢 Step 1: Create Employee Object

Go to:  
👉 Setup → Object Manager → Create → Custom Object  

Fill the details:

- **Label:** Employee  
- **Plural Label:** Employees  
- **Object Name:** Employee  

### ⚠️ Important:
For **Record Name**, choose:

- **Type:** Text  
- **Field Label:** Name  

---

## 🟢 Step 2: Create Fields

Go to:  
👉 Employee → Fields & Relationships → New  

Create the following fields:

### 1️⃣ Phone
- **Type:** Phone  
- **Field Name:** Phone__c  

### 2️⃣ Email
- **Type:** Email  
- **Field Name:** Email__c  

### 3️⃣ Salary
- **Type:** Number  
- **Field Name:** Salary__c  
- **Length:** 10  
- **Decimal Places:** 2  

---

## 🟢 Step 3: Create Tab

Go to:  
👉 Setup → Tabs → New  

- Select **Employee**
- Choose any tab style  
- Click **Next → Save**

---

## 🟢 Step 4: Apex Class

Go to:  
👉 Developer Console → File → New → Apex Class  

### ✅ Apex Code:

```apex
public class EmployeeController {

    public static void createEmployee(String name, String phone, String email, Decimal salary) {
        
        Employee__c emp = new Employee__c();
        
        emp.Name = name;
        emp.Phone__c = phone;
        emp.Email__c = email;
        emp.Salary__c = salary;
        
        insert emp;
        
        System.debug('Employee Created Successfully');
    }
    
    // Fetch Employee Records
    public static List<Employee__c> getEmployees() {
        return [SELECT Name, Phone__c, Email__c, Salary__c FROM Employee__c];
    }
}
```

## 🟢 Step 5: Run Apex Code

Go to:  
👉 Debug → Open Execute Anonymous Window  

Run the following code:

```apex
EmployeeController.createEmployee(
    'Keshav Kalani',
    '9518986776',
    'kalani15keshav@gmail.com',
    5000000
);
```
## 🟢 Step 6: Verify Output

Go to:  
👉 App Launcher (☰ icon)

Search for:  
👉 **Employee**

### Steps to verify:

1. Click on the **Employee** tab  
2. You will see the list of employee records  
3. Open the newly created record  

### Expected Output:

The record should display the following details:

- **Name:** Keshav Kalani  
- **Phone:** 9518986776  
- **Email:** kalani15keshav@gmail.com  
- **Salary:** 5000000  

### Result:

The employee record has been successfully created using Apex code and stored in the Salesforce database.
